Задачей агента поиска книг по событиям является нахождение книг по множеству событий, которое будет ему передано. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска книг по событиям. Единственным аргументом запроса является множество событий, на наличие которых пользователь желает осуществить поиск книг.

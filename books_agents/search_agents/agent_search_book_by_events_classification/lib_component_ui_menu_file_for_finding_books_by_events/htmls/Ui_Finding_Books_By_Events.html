Команда поиска книг по описанию событий предназначена для нахождения книг по указанным характеристикам событий. Единcтвенным аргументом команды является множество, включающее в себя требуемый класс события, роль и пол персонажа этого события. Результатом выполнения команды является отображение множества книг, содержащих указанные события.
